# SnapChess

SnapChess is a web chess app built for the `snapie.io` ecosystem.

It includes Hive Keychain login, game creation, invite links, time controls, rated and casual matches, and optional HIVE stake games.

## Tech Stack

- TypeScript
- Next.js
- React
- Hive blockchain integration via `@hiveio/dhive`

## Purpose

This project was made as a chess experience for `snapie.io`, using Hive accounts and Hive-based game flows.

## Development

```bash
npm install
npm run dev
```

Local app:

```bash
http://localhost:3000
```
